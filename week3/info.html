Fortune's Chao Garden Project

Made for Connections Lab Fall 2025 Midterm by Dae Kim


An fun interpretation of the Chao Garden from the Sonic video game series utilizes an 8ball API for "fortune telling". The user is assigned a chao that relates to the result given by the API.

Usually, I'm not a data person. However, for this project, I didn't want to make a one-off thing that I don't feel passionately about. I've always loved chao and thought it could be cute to do a project that would be wholesome.

Please check out the documentation for more information :D
